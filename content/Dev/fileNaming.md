1.  index.html 은 브라우저의 default 문서이다. home page 는 index.html 이름을 사용하면 된다.
2.  -(dash), \_(underscore)와 ~(tilde)를 제외하고 파일 이름은 알파벳과 숫자 등 문자로 이루어져야 한다.
3.  스페이스를 사용하지 않아야 한다.
4.  소문자를 사용해야 한다.
